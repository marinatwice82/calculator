Инструкция

Для простых операций "+", "-", "/", "*" сначало вводится первое число, затем нажимается кнопка с нужной операцией, потом вводится второе число и для получения результата необходимо нажать кнопку "=".

Для извлечения квадратного корня, необходимо набрать число и нажать кнопку "√". Результат сразу появится на экране (кнопку "=" не требуется нажимать). При попытке извлечения квадратного корня из отрицательного числа выведется на дисплей калькулятора сообщение "error".

Для возведения в степень, необходимо вначале ввести возводимое число и нажать кнопку "xу", затем ввести степень в которую нужно возвести число и нажать кнопку "=".

Для того чтобы сделать число отрицательным ( например: -9 ), необходимо после введения числа нажать кнопку "+/-". При повторном нажатии данной кнопки, число станет опять положительным.
